Wood Location - Test Page

This page tests all the functionality of the Wood Location application.

Functionality Tests

Manual Testing Checklist

Form Submission: Fill out all fields and submit the form. Check if you receive a success message.
Form Validation: Try submitting the form with empty fields. Check if validation errors appear.
Phone Formatting: Enter a phone number and check if it formats correctly with spaces.
Form Persistence: Fill out part of the form, refresh the page, and check if data is restored.
Tab Navigation: Click between "Contact Form" and "QR Code" tabs. Check if content switches correctly.
QR Code Generation: Go to the QR Code tab, enter a URL, and generate a QR code.
QR Code Download: After generating a QR code, click the download button.
Mobile Responsiveness: Resize the browser window or use developer tools to test mobile view.
Touch Interactions: On a touch device, test all buttons and form fields.
